# 带坐标轴	

包含的组件类型有：折线图、柱图、象形图、散点图等。

#### 数据格式
```json {3-5}
[
  {
    "x": "星期一",
    "y": "400",
    "s": "系列一"
  },
  {
    "x": "星期二",
    "y": "180",
    "s": "系列一"
  },
  {
    "x": "星期三",
    "y": "200",
    "s": "系列一"
  },
  {
    "x": "星期四",
    "y": "100",
    "s": "系列一"
  }
]
```
